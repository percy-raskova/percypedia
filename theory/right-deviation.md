# Right Deviation

:::{warning}
**This document is a stub.** Content has not yet been written.
:::

The right deviation, or opportunism, represents an error in revolutionary practice characterized by:

- Demanding nothing from the masses
- Coddling and patronizing the working class
- Abandoning revolutionary goals for reformism
- Tailing the masses instead of leading them
- Class collaborationism
